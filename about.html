Sooo... I started off with this idea ...

And that "idea" came while "marathoning" 24 on Netflix... I overheard the phrase terminate with extreme prejudice. Now this is neither the first nor last time I will ever hear that phase, yet it dawned on me to google it.

At this point, having gone so far in researching such an inane subject, I figured: why not create a logo? However, creatively speaking, this is a slippery slope because Obsessive Compulsive Creative Disorder starts to kick in, and it's just a matter of time before I'm in it for the long haul. Optimizing DRY philosophy (Don't Repeat Yourself) I created a "template" git repo, should I ever need to "repeat" this silliness. Thus I would have a point to get started and "let loose the creative juice."

git repo created, Twitter bootstrap integrated....

Now finally I'm going to town on this rando "idea"... add the logo, add some ipsum... I remember that I have this awesome Halloween picture of me in a gas mask which would work great so I throw that up. This is starting to grow legs. I go back and reassess the color schematics and start making the red more prominent. Now it's time to get some fonts cracking so I find two complimentary Google Fonts.

I still have this lingering feeling of "What is the point of this site?" Looking back from a UX perspective, I realize that the site is overwhelmingly dark, with accents of red... Two major design faux pas ... But I figure: run with the theme and don't turn back. I've also integrated bootstrap so my button should do something, I've styled that button, yet what is it going to do? Keeping with the theme of "no turning back"...

The button should nuke the whole page...
